# GitHub
The project is running on GitHub, where the structure of the GitHub is as follows:

- Organization - *GIRAF*
    - Repos - *All the different parts of the projects*
        - Weekplanner
        - Api-client
        - Web-api
        - .github
        - ...

## Getting started
In order to be able to work on the project, all of the students need to be registered as either members or owners of the organization
